# ViewedFilms

An application in which you can record interesting movies.


## Features

- UIKit
- MVVM
- Core Data
- OOP
- SOLID
- GitFlow
- Light/dark theme

## Opportunities

- Name
- Photo
- Release date
- Rating
- Video teaser
- Description
